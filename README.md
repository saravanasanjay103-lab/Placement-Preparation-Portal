# Placement Preparation Portal

## Overview
A backend web application developed using Java and Spring Boot to help students prepare for placements.

## Features
- Student Management
- Aptitude Module
- Coding Questions
- Result Management
- Progress Tracking

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Postman
